**ClipboardPro** is a premium, high-performance native Windows clipboard manager and productivity suite. Designed for developers, designers, and power users, it helps you manage your clipboard history, automate text templates, and share items seamlessly across your local network.

---

## ✨ Key Features

### 1. 🗄️ Multi-Format Clipboard History Vault
* **Access Anywhere:** Press `Ctrl + Alt + V` to summon the dashboard.
* **Smart Categorization:** History is auto-categorized into *Texts*, *Images*, *Colors*, *URLs*, and *Files*.
* **Instant Search:** Powerful fuzzy search helps you find copied items in seconds.

### 2. ⚡ Cursor-Snapping Quick Paste Bar (Mini Mode)
* **Arrow & Enter Control:** Press `Ctrl + Shift + V` to display a horizontal quick-paste bar directly beside your active cursor. Select and paste without losing context.

### 3. 🗃️ Drag & Drop Temporary Shelf (macos Style)
* **Yoink Docking:** A floating edge bubble lets you drag, drop, and collect multiple files, images, or snippets, and later drag them out together.

### 4. 🧠 Smart Format Detection & Utilities
* **JSON Prettifier:** Formats minified JSON on the fly.
* **Color Preview Badge:** Color codes (HEX, RGB) are shown with colored badges.
* **URL Web Title Crawler:** Displays page titles (e.g. "Google") instead of messy links.
* **API Key Obfuscation:** Mask AWS, Stripe, and sensitive keys dynamically.

### 5. 🚀 Keyboard Text Expander (Snippets)
* **Quick Expansion:** Create shortcut triggers (e.g. `:email`) that automatically expand into long text templates (e.g. `support@clipboardpro.com`).

### 6. 🌐 Peer-to-Peer LAN Sharing
* **Instant Sync:** Copy an item on one PC and instantly paste it on another connected to the same LAN/Wi-Fi network.

## 💾 Installation

### Method 1: Direct Download (Recommended)
Get the latest installer executable directly from our **[Releases Page](https://github.com/mitul002/ClipboardPro-Official/releases/latest)**.

### Method 2: Winget
Once indexed, you can install ClipboardPro via Windows Package Manager:
```cmd
winget install ClipboardPro
```

### Method 3: Chocolatey
Once approved, you can install via Chocolatey:
```cmd
choco install clipboardpro
```
